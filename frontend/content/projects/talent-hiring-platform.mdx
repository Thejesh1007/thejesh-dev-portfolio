## Overview

Talent Hiring Platform is a full stack application designed to simulate
a real hiring workflow where recruiters can create jobs and candidates
can submit applications.

## Problem

Many hiring platforms lack structured role separation between recruiters
and candidates. This project focuses on implementing a clear workflow
for job creation and application management.

## Architecture

Frontend: React + Vite  
Backend: Node.js + Express  
Database: PostgreSQL  
ORM: Prisma  

## Key Features

- Role-based authentication using JWT
- Recruiter dashboard for job management
- Candidate job application workflow
- Admin monitoring capability

## Challenges

Designing relational schema for users, jobs, and applications while
ensuring clean API structure and validation.

## Future Improvements

- Email notifications
- Job search filters
- Application analytics
