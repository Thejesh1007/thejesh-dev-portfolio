## Overview

The Talent Hiring Platform is a full-stack SaaS-style web application designed to connect recruiters and candidates through a structured job application workflow.

The system implements role-based access control, authentication, dashboard segregation, and a production-ready backend architecture.

---

## Architecture

The system follows a layered architecture:

- React (Vite) frontend
- Express REST API backend
- Prisma ORM
- PostgreSQL database
- JWT-based authentication
- Role-based authorization (Candidate, Recruiter, Admin)

The frontend communicates with the backend via REST endpoints secured by JWT.

---

## Key Features

- Role-based authentication
- Recruiter dashboard for job posting
- Candidate dashboard for job applications
- Admin monitoring capabilities
- Secure password hashing
- Production deployment on Render
- Clean SaaS dark-themed UI

---

## Engineering Decisions

Prisma was chosen for type-safe database interaction and maintainable schema evolution.

JWT was used for stateless authentication to enable horizontal scalability.

The system emphasizes separation of concerns:
- Controllers
- Routes
- Middleware
- Validation
- Error handling
